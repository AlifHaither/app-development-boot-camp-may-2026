# MExpense

MExpense is a simple Flutter expense tracker app built for the Miftah Coding Bootcamp.

## Features
- Add income & expense
- Transaction history
- Search & filter
- Local data storage
- Analytics dashboard
- Editable profile

## Tech Stack
- Flutter
- Provider
- SharedPreferences

## Run Project
flutter pub get
flutter run
